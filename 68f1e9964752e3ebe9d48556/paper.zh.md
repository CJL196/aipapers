# 摘要

视频是异步学习的便利资源，但缺乏同步课堂中存在的人际互动。由于缺乏社会联系，孤立的视频学习者在情感、行为和认知参与度上表现较低。本研究提出了“脚本化替代对话”（SVD）技术，通过让学生观察虚拟角色（教学助理和学生）之间围绕视频进行的脚本化对话，从而让学生体验一种伪社交的经历。我们进行了参与式设计研究，以推导SVD的设计指南。研究结果表明，需要区分虚拟组件，并给予学生对对话节奏的控制。随后，我们实现了SVD的交互原型，并在一个非社交的直接学习基线下进行评估（N=40）。结果显示，SVD与基线的偏好存在极化（40人中有25人偏好SVD；没有中立偏好），而偏好SVD的学生在情感和行为参与度上显著高于基线。

# 1 介绍

教育视频在孤立学习环境中非常有用，这种环境随着以学习者为中心的教学法的流行、教育视频平台（如慕课）的可用性以及在COVID-19流行期间转向远程学习而变得更加普遍。然而，讲座视频存在教学上的局限性。首先，它们仅展示讲师，没有任何人际互动，例如其他学生的问题或同伴讨论。研究表明，学习者可能会感到孤立，这会影响他们的学习动机，导致“当下”的行为、情感和认知参与度下降。参与度的降低是一个问题，因为这会削弱学习者的学习成果。缺乏社交互动和参与手段为丰富教育视频周围的学习环境提供了机会。

有几种方法可以将社交元素添加到观看视频的体验中，但它们存在缺陷，削弱了基于视频学习的优势。例如，学生和教师或助教之间的同步共同观看会牺牲学生在自己选择的时间和地点学习的便利性。同步课堂的录音可以让学生通过他人的互动进行旁观学习，但对于录制课的重复使用可能存在隐私问题。此外，现场讲座中问题的质量和数量并不能得到保证，学生所提出的观点可能并不总是有利于旁观学习者，提出的问题数量可能太少而无法产生积极效果。超越同步方法，诸如弹幕、YouTube评论或TrACE等可见和带时间戳的评论功能可以促进一些互动，但社区驱动的评论往往噪声很大，需要手动审核以确保其相关性和质量。相比之下，我们的目标是设计和测试一种学员参与的方法，其中增强的内容由教师精心策划，有利于学习者的便利访问。

在本文中，我们提出了脚本化的替代对话（SVD）来增加学习者的参与度。SVD是一种视频增强模式，学习者目睹增强内容作为虚拟角色（如助教、同学）之间的脚本化教学对话，仿佛他们在Zoom课堂上共同观看视频。我们借鉴了社会建构主义，通过互动来提高参与者（在本例中是虚拟角色）的参与度。而替代互动理论支持这种好处可以传递给目睹对话的替代学习者。正如图1所示，虚拟角色之间的互动被脚本化，仿佛他们第一次一起在视频会议平台（例如Zoom）观看讲座视频，并在此过程中参与与视频相关的活动和讨论。

尽管SVD利用虚拟角色，但应注意SVD与基于代理的方法不同，例如辅导/教学代理、可教授代理和同伴代理。使用SVD时，学习者并不与任何虚拟角色互动，也不会以任何方式影响其对话；他们所有的参与都将通过观察虚拟角色的互动间接进行。SVD的脚本化特性使得讲师能够完全控制对话内容的有效性和全面性，例如，他们可以包括常见问题和幼稚问题以及学习者可以从中学习的常见错误。相反，虚拟代理基于学习者的行为和表现来模拟对话。换句话说，基于代理的方法侧重于对话生成，而SVD则侧重于对话呈现。

我们旨在设计SVD体验——如何呈现对话和角色以达到最佳效果——并考察其在引发学生参与方面的有效性。由于SVD是一个新概念，结合了多角色设置和替代设置，我们无法根据现有方法的文献推导其设计要求或预测其效果。我们需要教师对其吸引学生的策略的反馈，以及学生对SVD概念的需求。推动我们对SVD进行研究的问题是：我们如何设计SVD，以提高孤立学习者的情感、行为和认知参与？

为了回答研究问题，我们进行了两阶段的研究。在第一阶段，我们通过与六位大学讲师和三位经验丰富的助教进行参与式设计研究，得出了SVD的设计指南。这些助教和讲师在课堂上使用讲座视频方面有经验。我们确定了三种构成有效SVD的人际互动类型，并为SVD制定了三个设计指南。基于这些发现，我们设计了SVD的初步原型，以便从学生那里获取反馈，然后最终确定设计。在第二阶段，我们实施了一个互动原型并进行了评估研究。我们对40名本科生进行了评估研究，将SVD与基线（非社交的、视频中的弹出测验）进行了比较。我们发现对SVD与基线的偏好是两极分化的（没有中立偏好），影响偏好的潜在因素包括学习者对社会联系的个人需求、对虚拟性的感知以及学习策略。我们还发现，偏好SVD的学生在情感和行为参与度方面显著高于基线。

本论文的主要贡献是将奇异值分解（SVD）作为一种新颖的教育视频增强方式的概念性贡献。在此贡献中，有三个次要贡献：

(1) SVD的设计指南：我们提出了三条设计指南，涉及展示、虚拟角色的人物设定和学习者对SVD的控制（第3.3节）。  
(2) SVD的设计产物：如图1所示的设计基于理论文献和设计指南（第3.4节）。  
(3) SVD的实证评估（以世界历史为例学科领域）：我们的结果显示，SVD增加了学习者的情感和行为参与，而没有牺牲认知参与，前提是学习者重视人际互动且对虚拟对话感到舒适（第5节）。这一结果也是代理互动理论可以应用于观察纯虚拟角色之间对话的实证证据。

# 2 相关工作

# 2.1 参与的定义及相关理论

在这一部分，我们将定义本研究中使用的参与概念，并总结与参与相关的两种理论，这两种理论是SVD概念的理论基础：社会建构主义和替代性交互理论。

参与是一个具有多重定义的术语，并在许多上下文中使用。在本研究中，我们采用了Fredricks等人提出的学术参与的定义，该定义由三个构成部分组成：行为参与、情感参与和认知参与 [22]。我们选择这个定义有两个原因：(1) 这是教育研究中广泛使用的定义，并且与用于测量参与的工具相关；(2) 因为它涵盖了与同龄人的社会参与，这与我们的研究目标一致 [1]。

情感参与包括在与课堂材料、教师和同学互动时的满意度、享受、兴奋、兴趣和好奇心。

行为参与包括学生的注意力、专注力、坚持不懈，以及在同步课堂环境中参与课堂活动。  
认知参与包括浅层参与，例如识别重要主题和记忆，以及深层参与，例如构建概念、应用理论和评估知识的实用性。

这个定义可以应用于广泛的教育背景，然而，在本文中，我们特别关注情境或“瞬间”的参与。这指的是在短时间内特定情境下的参与，这与关注学生在一个课程、一个学期或一个学年中的长期参与的参与形式相对立[64]。

第一个理论，社会建构主义，指出“知识是在环境中（心理交互上）与他人共同建构的。[..] 个体学习实际上是知识创造的产物，这发生在合作进行时，知识本身在环境中被共同创造”[67]。该理论已被应用于许多教学方法和界面，以通过与真实和虚拟同伴或更有知识的他人（MKO）互动来提高学生的注意力和学习收益[21, 55, 61]。（MKO指的是任何“能力更强”或“具备更高认知能力”的人，相对于学生而言[70]。）在社会科学或艺术科目中，这种互动也可以通过共享经验促进情感参与[30]。然而，由于这种情感效应尚未在其他学科中普遍适用，我们将仅应用此理论来激励与SVD的行为和认知参与的增加。

第二种理论是替代互动。这一理论源于社会认知理论，解释了人类如何通过观察学习。它指出，“当一个原本被动的学生（替代互动者）积极观察、吸收并处理其他学生之间及其他学生与其教师之间的互动时……[他们]可以享受到与直接互动者所获得的收益基本相当的好处。”这种方法通过知识的转移来增加学习收益，尽管这种收益可能低于直接学习。研究表明，替代学习还可以增加情感参与感，使学习体验更加愉快，从而转移直接学习者的热情和乐趣。在这项研究中，我们使用这一理论来激励概念设计，并假设社会建构主义对真实学生的益处。

本文展示了这些理论的实际实例，体现在设计界面功能以增强参与感上。（SVD设计的理论基础将在第3.1节详细解释。）它还提供了一个实证案例，表明旁观者可能从观察纯虚拟角色的互动中获得益处。

# 2.2 增加社交互动和视频观看体验中的参与感

有许多方法可以添加社交组件，使观看视频的体验更加愉快。在教育环境中，有效力代理人和带时间戳的评论的尝试。这些方法从一个界面开始，该界面通过文本消息和动画面部表情来激励学生 [13]，到一个能够解读学生行为并自适应鼓励他们参与小组学习的机器人 [2]。有效力代理人的问题在于它们经常为特定目的或学科而设计，因此教师使用起来可能会很困难，对学生来说也会很昂贵。此外，关于有效力教学代理人的伦理问题，涉及数据隐私、情感隐私以及人类与人工智能之间虚拟关系的其他影响 [51]。另一方面，带有时间戳的文本评论，例如弹幕、YouTube评论 [20] 或 TrACE [18]，都是从所有以前的观看者那里收集而来。来回的评论有助于促进教师与学生、学生与学生之间的交互以实现功利目的 [9]，而享乐性和社交性评论则帮助减少学生的孤独感 [38]。然而，社区驱动的评论往往噪音较大，需要进行实际的管理以确保相关性和质量 [32]。尤其是在弹幕的情况下，评论在屏幕上飞过，事实上会分散学生对视频的注意力，并需要更高的认知负荷 [71, 72]。

在非教育环境中，反应视频和共同观看会议是为视频观看增加社交存在感和互动的常见方式。反应视频（包括表情包和GIF反应）通过情感元目击来运作，观看其他人的情感反应可以给空间上距离遥远的观众带来亲近感。制作反应视频也可能成本高昂，因为这需要真实的人提前观看内容并录制自己。此外，还存在情感偏差的风险，即反应者与观众对内容反应的不一致可能导致对体验的厌恶感。最后，在共同观看会议中，间歇时的聊天已被证明可以使体验更加愉快和吸引人。弹幕评论也可以在共同观看会议中使用，尽管它们可能会造成视觉混乱。对于在线学习来说，共同观看会议的限制在于它们需要多个观众同步参加，否则就降低了可以异步观看的讲座视频的便利性。

此外，研究人员探索了视频制作策略和附加系统，以提高孤立学生的瞬时参与感，而不利用社交互动。不同类型的视频视觉效果和旁白已显示出对行为参与的影响[12]。注意力线索，例如声学[28]和视觉线索[29]，已被用于在视频的特定时刻增加学生的参与度。在此基础上，Kasperiuniene等人的关于基于虚拟形象的讲座的研究表明，复杂的多模态线索比单一模态的线索产生更多的参与感[34]。然而，这些线索仅能唤起行为参与感，并偶尔会引发情感参与感（如果它们具有美学吸引力），但并未在认知上吸引学生。许多研究采用了ICAP（互动、建构、主动和被动）框架，并根据学生的学习行为调整内容或学习活动，以提高他们的认知参与[15, 63, 68]。基于代理的方法，例如辅导和可教授的代理，也在改善认知参与方面普遍存在，作为提高学习收获的一种手段[8, 39]。ICAP驱动和基于代理的方法与共情代理和弹幕评论在信息隐私、缺乏教师控制和潜在的噪声行为信号方面存在类似问题。

与这些解决方案相比，SVD 让教师可以控制所有互动，并确保它们对学生是合适和有帮助的。在学生这一方，增强的讲座视频随时可供访问。此外，由于虚拟伙伴与实际学生没有互动，这一解决方案规避了智能代理的伦理问题。

# 2.3 在线替代学习

现有的孤立在线学习者的替代学习环境可以根据直接互动者的特征分为四类。首先，是一种替代学习环境，目标学生观看一位真实讲师与多位真实学生之间的对话。最显著的例子是Zoom课堂的录音。一项大规模研究表明，这种录音中的对话显著提高了语言学习者的认知参与度。其次，是一种环境，目标学生观看一位智能虚拟讲师与一位真实学生之间的互动，这种设置通过一个名为GuruTutor的工具进行了探讨，研究表明它能减少思维漂移并提高行为参与度。第三，Nugraha等人探讨了一个讲师与一位讲师创作的虚拟学生之间的设置。它允许讲师自己创建替代学习场景，避免安排学生的需求，并比虚拟代理更好地控制对话内容。第四，目标学生通过其他在线学生的众包问答进行学习的设置。这一选项在一个名为YouTute的工具中使用，该工具基于其他学生的回答为现有教育视频提供替代学习体验。尽管替代学习在学习收益方面显然有益，但大型教育平台，如MOoCs，并没有采用这一概念的任何功能。

SVD 是这些不同类型的结合。它使讲师对对话内容具有完全控制权，并扩展互动，包含多个虚拟角色，以增加与同步课堂环境的相似性。我们还将视频的范围从讲师自己的讲座扩展到任何在线视频，以便 SVD 可以应用于教育视频平台，增强和定制现有内容。

# 3 设计脚本化的替代对话

在本节中，我们描述了研究的第一阶段。该阶段的目标是制定SVD的设计指南。我们的设计基于理论调查和参与式设计。我们的研究聚焦于SVD的四个方面：

要模拟的课堂人际互动类型，  
模拟互动的呈现（例如，现实感和互动隐喻的程度），  
学习者对模拟互动的控制级别，以及，  
虚拟角色的个性和行为。

# 3.1 理论基础和概念设计

在本节中，我们将解释社会建构主义和替代性交互理论（在第2.1节中总结）如何激励我们的SVD设计，以及为什么我们期望该设计能够有效提升学习者的参与度。

SVD的概念设计至少包含一个MKO角色（教学助手、更有能力的同伴或讲师）和多个虚拟学生。由于该角色的行动完全由真实讲师控制，真实讲师是MKO，因此我们可以假设该角色的行为对虚拟学生角色来说是MKO的行为。这些角色将共同观看原始视频内容，并就视频主题进行讨论。学习者将通过观看原始视频和观察围绕视频的互动来学习。

在第一步中，我们应用社会建构主义，该理论认为与MKOs的社会互动会导致更高的行为和认知参与 [67, 70]。根据社会建构主义，我们期望虚拟学生对学科内容有更高的行为和认知伪参与。

在第二步中，我们应用替代互动理论来建模虚拟学生的伪体验如何转移到学习者身上[59]。我们假设学习者将表现出更高的行为和认知参与，这将是通过替代学习从虚拟学生转移来的好处。需要注意的是，替代互动理论并没有明确规定行为和认知参与是否属于可转移的好处，以及在直接互动者完全是虚拟的情况下是否适用。因此，我们的研究有助于填补这一空白。我们研究对该理论的影响将在第6.1节进一步讨论。

除了学习的认知和行为方面，该理论还被应用于虚拟实体与人类之间的情感传递 [23, 36]。我们预计观察虚拟角色的乐趣和热情也可能有助于提高学习者的情感参与度。

# 3.2 参与式设计

在前一部分的理论基础上开发了基于SVD的概念设计后，我们进行了参与式设计研究，以收集教师对SVD概念的印象和意见。我们与大学教师和助教进行了研究，以收集有关实际课堂中人际互动的信息。我们基于Scheepmaker等人的论文[54]的研究方法开展了参与式设计研究。每个会议包括关于他们的教学和视频制作经验的访谈、一个头脑风暴活动以引出他们在课堂上看到的互动，以及一个协作设计活动（参与者与研究人员之间）。在头脑风暴活动中，研究人员促使教师列出他们在课堂（在线和离线）中看到的尽可能多的互动。然后，教师向研究人员解释每个互动的重要性，并头脑风暴他们认为如何将这些互动复制给观看课堂讲座视频的学生。在设计活动中，我们提前准备了低保真度的SVD设计草图，以促使参与者对其进行批判和扩展。

我们通过在大学内部进行滚雪球抽样，利用实验室和部门的联系招募参与者，招募来自多个学科的讲师和助教，以评估该工具的跨学科潜力。参与者包括来自科学、社会科学和人文学科的七个不同部门的六名大学讲师和三名助教（P1-P9）。每位参与者都以异步方式教授过1-3门课程，并同步提供相同的课程，以便比较讲座视频缺失的社交成分如何影响他们的教学策略和学生的参与感。每位参与者的时间和努力得到了65美元的补偿。

根据文献综述和参与设计的设计指南，我们创建了SVD的初步原型。我们通过Prolific招募了10名本科生（P10-P19）。参与者需为流利使用英语的本科生。在研究期间，他们观看了一段关于第一次世界大战或伪随机数生成器机制的视频，并与SVD进行了增强，随后提供了关于设计及SVD可能带来益处的上下文的反馈。所有参与者均获得了9.50美元的报酬。

3.2.1 数据分析。我们从参与式设计研究中转录并分析了数据，包括访谈文本、头脑风暴记录和会议笔记，以制定潜在的设计指南。在初步评估研究中，我们计算了对 SVD 各组成部分（例如，聊天消息、TTS 语音、表情符号反应和第 3.3.1 节详细说明的三种交互类型）感知效果的平均李克特量表评分，以确定设计中的问题领域，然后分析他们的定性评论以了解问题的性质。最后，我们完善并补充了初始设计指南，以考虑学生的需求和对设计的感知，然后最终确定 SVD 的设计。

# 3.3 发现

3.3.1 课堂互动构成有效的SVD。我们的SVD设计包括三种参与者反复提到的互动类型，这些互动类型有助于增强学生的参与感，并且可以在在线课堂环境中重复进行。这些互动类型是：

(1) 问答。这是一个信息互动，有助于学生深入理解课程内容。它对所有学生都有益，包括那些在提问时不够积极或自信的学生，他们可以通过旁观提问学生与讲师的问答来受益（P1、P2、P3、P5）。讲师也可以主动发起这种互动，通过提问或提示学生基于某个主题或概念进行头脑风暴来测试学生的知识（P4、P7）。讲师的问题非常重要，因为它促使学生验证和加深对课堂内容的理解。此外，学生可以对自己在课堂中的位置有一个了解，如果他们知道自己的错误是普遍存在的，他们会感到安心（P2）。

（2）社交闲聊。这是一种社交和情感互动，旨在“吸引学生的注意力，提高课堂的活力”（P9），并在课堂内建立师生之间以及学生之间的融洽关系（P8）。它通常包括问候、非学术性对话和其他交际互动。通常发生在课程开始和结束时。我们从初步评估中发现，在SVD中，社交闲聊可能会感觉单调无关（P11，15，18，19），因为学生并没有将角色视为可以交往的真实人物。因此，我们将社交闲聊保持简短，并与视频的学习内容相关。

（3）非语言反应。用于社交和情感目的的非语言反应（例如，贴纸和表情符号）在COVID-19大流行期间变得普遍（P5）。它很有价值，因为它可以代替情感表达（例如，微笑、笑声、翻白眼），这些都是教师和同龄人需注意并与“看起来有些疲惫的学生”或“某个概念或细节尚未理解”的学生共情的重要线索（P9）。

3.3.2 设计指南。通过参与式设计研究，我们提出了三个设计指南（DG）。在这里，我们详细说明这些指南以及它们如何支持我们在3.4节中呈现的最终SVD体验设计。

DG1. 平衡虚拟性和现实主义 SVD的外观过于真实是一个问题。这是因为SVD的脚本是由讲师提前编写的，因此包含经过深思熟虑的问题，这些问题与真正学生在同步课堂上可能提出的问题不同且更具挑战性。如果角色的外观与提出此类问题的角色相似，学生可能会被误导，将自己的表现与角色进行比较（P7）。因此，建议SVD中角色的视觉表现明显具有虚拟性，以加强SVD是一种仅用于模拟真实课堂的脚本。然而，我们需要注意的是，对于对话的语音叙述，合成的虚拟质量可能也是一个问题。特别是，我们的初始设计实现了使用文本转语音（TTS）为虚拟角色进行语音叙述，但许多参与者（P12，P15，P18，P19）对合成的声音不满意，因为他们认为声音“像机器人”，“令人烦恼”，且“令人毛骨悚然”。这种对声音不自然的反感让参与者在情感上 disengage，从而影响他们的情感参与。

DG2. 创建具有个性和多元身份的 relatable 角色 学习者能够“与角色产生共鸣，以便他们能够更深入地参与[与角色的互动]”是很重要的（P5）。为了吸引广泛的学习者，角色的身份，如种族和性别，必须多样化和包容性（P4）。P8还指出，除了性格之外，为角色赋予可区分的特征，如爱好或彼此之间的关系，可能会很有趣，因为这些话题使学生能够在真实课堂上进行讨论和建立联系。

我们应该警告这些三个方面可能相互冲突，实现它们之间的平衡是一个具有挑战性的设计问题。例如，P4表示我们必须小心避免刻板印象，当包括其他特征时，例如课程的熟练程度。此外，使角色能够代表多样的种族、性别和其他特征意味着学习者可能无法平等地与所有角色建立联系。

DG3. 允许学习者自主调整对话的节奏。SVD的初始设计是视频播放，犹如Zoom课堂的录制。在初步评估中，参与者发现了这个设计的两个问题。一方面，一些参与者报告称互动太慢，因为“人类可以花很长时间来向你解释一个想法。”（P13）然而，另一些参与者觉得体验是被动的——在看到答案之前，他们并没有感到有必要或有足够的时间积极思考问题，正如P11所评论的：“替代对话并不要求你思考[...] 有一个停顿，但即便如此，[虚拟]学生的回答真的非常快。[比我快得多。] 所以，我只是读了他们的答案。”这表明学习者有不同的阅读速度和吸收对话的方式（仅仅阅读与积极思考）。因此，SVD的界面应该提供一个方便的方式，让学习者以他们所偏好的节奏浏览内容。

# 3.4 设计

在本节中，我们将描述在纳入设计指南后SVD的最终设计。概念模型是一个Zoom教室，这体现在观众窗口、视频和聊天窗口的布局和风格中（见图1）。

在DG1之后，当视频暂停时，虚拟角色之间的所有互动以文本消息和表情符号的形式进行，从而使它们在时间和空间上与真实组件，即原始视频区分开来。时间上的分离在进度条中得到了强调，其中连续的视频和离散的SVD组件分别用线条和点表示。我们还对角色的头像应用了卡通效果，以突出互动的虚拟性。最后，我们强调，学习者在使用该界面之前会被告知对话是完全编写的，并不反映或基于任何真实的课堂。

DG2. 影响我们的脚本创作过程。角色个性有三个显著特征：性别、种族和互动频率。我们通过角色的头像和名字来表现他们各种性别表达和种族身份。我们选择使用文本回应的频率而非表情符号，因为在文本场景中它比个性特征更易于识别。一些虚拟学生会在聊天中输入较长的答案，并频繁提出后续问题，但有些只会输入短语或单词，不询问后续问题，并在可能的情况下使用表情符号。我们也小心不让虚拟学生表现得明显优于或劣于其他学生，因此我们不创造或执行与学业表现相关的任何刻板印象。

最后，根据DG3，我们将SVD制作成一个互动界面，学习者必须点击“点击继续”按钮，以便每条信息逐个出现。我们还允许学习者通过进度条中的点来来回导航，不同对话之间也可以导航。每个点都是可点击的，可以引导学习者到特定的文本气泡或表情符号反应。通过这些功能，学习者可以以最有利于他们学习的方式自主调整学习节奏。

# 4 评估方法

为了评估第3节中的设计，我们对SVD与基线（一种直接学习的非社交视频增强方法）进行了比较评估。根据第2.1节中提出的参与度定义，我们在本研究中测量了三种参与类型：情感、行为和认知。此外，我们还测量了学习收益，这是所有三种参与类型的预期后效。

# 4.1 基线系统和实验材料

我们选择视频中的测验作为基线，因为它们是学生熟悉的活动，并且可以通过三种方式与SVD进行公平比较。首先，它是一种直接学习和非社交的设置，这与SVD的替代和社交设置形成对比。其次，它在所传达的学习内容量和增强内容的时间上与SVD没有区别。最后，SVD和视频测验对讲师所需的输入和努力程度相似。（我们在第6.5节讨论自动将测验提示转换为SVD脚本的可行性。）为了实施基线条件，我们采用了EdPuzzle 5，因为它的学习界面简洁直观，而其他类似工具则有许多额外功能，可能会分散学习者的注意力并干扰研究的目标。此外，它是一个稳定且易于访问的网络应用程序。EdPuzzle的示例截图如图2所示。

为了选择原型的视频，我们遵循了四个标准：（1）视频必须在同一主题上以确保主题一致性；（2）视频长度必须相同，并且在与EdPuzzle和SVD增强后，时长应为6.9分钟，以最大限度地提高参与度[27]；（3）视频不应需要其他视频作为先决条件；（4）视频不应是广为人知的话题。我们在可汗学院6的在线视频集合中搜索，以找到符合标准的视频。最终，我们选择了一段关于第一次世界大战7的视频和一段关于拿破仑战争8的视频。

为了编写增强内容的脚本，我们从每个视频中剪辑了三个部分，并用内容完全相同的增强内容替换了它们。我们根据DG2（3.3.2）撰写脚本。我们通过将两个视频的对话结构化为相同的形式来促进公平性（例如，问候和告别对话的内容、虚拟学生错误回答的次数、教师在开始讨论之前的视频回顾长度）。我们还为每个对话编写了不超过15个聊天气泡或表情符号的脚本，以确保对话不会使学习者从视频中失去参与感。然后，我们创建了前测和后测问题（前测和后测是相同的）。每个测试由五个记忆问题和五个应用问题组成，以捕捉浅层和深层认知活动。我们控制了两个视频的测试，从增强内容和原始视频中抽取相同数量的问题，以及相同数量的选择题和简答题。

# 4.2 实验设置与程序

我们设计了一项 $2 \times 2$ 的因子实验（条件：Edpuzzle 和 Vicarious Dialogues；视频：第一次世界大战和拿破仑战争）。9 种条件是组内变量。条件 x 视频的配对是组间变量。我们控制了在不同条件下，问答增强内容在视频中出现的时间和位置。我们还控制了视频中的 SVD 设置（一个助教和三个学生）。参与者观看了一段带有 SVD 的视频，然后观看了另一段带有 EdPuzzle 的视频。视频和增强方法的配对是相平衡的。增强方法呈现的顺序也是相平衡的。

我们测量了4个依赖变量。在观看每个增强视频之前和之后，参与者完成了相同的测试（附录A）以测量他们的学习收益。在完成后测之后，参与者填写了一份问卷（附录B），该问卷使用7分Likert量表来测量他们的行为、情感和认知参与度。参与者在开始时被告知每个视频的前测和后测包含相同的问题。在实验结束时，参与者填写了一份后任务调查（附录C）。在调查完成后的24小时内，我们发送了独特的后续问题，以详细解释他们的调查回应。每位参与者的补偿为1230美元。

# 4.5 数据分析

我们使用马克斯和卡明斯提出的规范化变化公式计算了学习增益，因此结果范围从-1到1。然后，为了考虑其他变量可能产生的影响，我们拟合了混合效应模型来分析这四个因变量。

学习增益（LG）结果的标准线性回归模型形式为：

$$
L G = \beta _ { 0 } + \beta _ { 1 } \times S V D + \beta _ { 2 } \times W W I + \gamma _ { j } \times p a r t i c i p a n t _ { j } + \epsilon _ { i j }
$$

其中SVD和WWI是代表参与者观看的干预和视频的二元变量（例如，$\mathrm { S V D } { = } 0$表示EdPuzzle条件，而$\mathrm { W W I } = 0$表示拿破仑战争视频），它们的系数表示固定效应。每个独特的参与者$j$都给定了自己的系数$\gamma _ { j }$，以考虑因观察依赖性而产生的随机效应（因为条件是组内变量）。该模型还包括一个误差项$\epsilon _ { i }$，表示模型中其他变量未解释的值的变异。$\beta _ { 1 }$是从EdPuzzle到SVD的斜率，因此为了验证参与者在SVD上相比于EdPuzzle是否有显著更高的学习收益，我们检查了$\beta _ { 1 }$参数是否为正并且显著不同于0 $\left( \mathrm { p } < . 0 5 \right)$。另一方面，显著且负的$\beta _ { 1 }$将意味着参与者在EdPuzzle上有显著更高的学习收益。

关于参与评分，三个参与评分均为汇总的7级李克特量表得分：行为参与得分范围为4到28；情感参与得分范围为7到49；认知参与得分范围为10到70。尽管原始响应为李克特量表，但汇总的李克特量表得分可以视为连续的，因此我们也可以对它们使用线性混合效应模型。该模型的解释与学习收益的解释相同。

最后，我们将参与者在任务后调查和后续访谈中的反馈与他们的参与度评分和学习收益进行了三角验证，以验证我们设计指南的效果，明确SVD的好处，并推测SVD的设计如何与参与者的学习风格和偏好相互作用。

（2）不适（ProBaseline）与适应（ProSVD）的角色虚拟性：偏爱SVD的参与者对虚拟角色及对话剧本化的知识感到舒适，而偏爱EdPuzzle的参与者则觉得虚拟互动让人感到不自在且分散注意力。这种对比在社交闲聊的反馈中尤为明显。
（3）不愿意（ProBaseline）与愿意（ProSVD）不断与系统互动：偏爱EdPuzzle的参与者觉得逐条点击信息很乏味，分散了他们对互动内容本身的注意力。

第一个原因是，SVD可能迫使参与者在微观层面上自我调节学习，使他们将注意力集中在学习任务上，而不允许他们分心。他们必须逐一点击每条消息。因此，这鼓励了参与和参与，而不是作为“被动旁观者”（P58）。这种控制也允许那些已经了解内容或觉得对话偏离主题的人随意加快互动：“你不会关闭并睡着，因为你可以跳过它。”（P56）值得注意的是，尽管这个原因与第三个偏好因素（愿意不断与系统互动）密切相关，但那些不愿意与系统互动的人仍可能会发现，系统吸引了他们的注意力和专注力，即使他们并不享受这个体验。

“在虚拟对话中，尽管我无法在聊天中实际回应，但我觉得自己更像是对话的一部分，在我的脑海中思考即使是针对其他角色的问题我会给出的答案。对于EdPuzzle的问题，我对详细阐述和深思我的答案以及我所给的答案的欲望较少。我会写下简单的1个或2个单词的答案，以便快速通过讲座并检查我的知识是否正确。”

# 5.3 认知参与

我们发现其他任何认知参与评分的比较没有显著差异，无论是正面还是负面。

ProSVD: $\beta _ { 1 } = 1 . 3 7 8 2$ , $\mathrm { p }$值 $= 0 . 1 1 9$ (不显著) ProBaseline: $\beta _ { 1 } = - 2 . 2 1 4 3$ , p值 0.197 (不显著) ALL: $\beta _ { 1 } = 0 . 0 3 6 3$ , p值 $= 0 . 9 6 5$ (不显著)

这在理论基础上是可以预期的。像 EdPuzzle 这样强制学生思考的直接学习，通常比像 SVD 中的替代学习更具认知参与性。然而，模拟对话很可能比单向知识传递方式增加认知参与，如社会建构主义所述，因此两种系统最终未产生显著差异。

我们根据参与者的定性反馈发现了三个有趣的观察，这些观察可以解释 SVD 如何影响参与者的认知参与：（1）替代学习减轻了表现焦虑；（2）自我节奏有助于消除认知过载；（3）学生在 EdPuzzle 上更积极地思考问题，而在 SVD 上更积极地思考答案。

由于替代学习，SVD减少了与同步课堂相关的表现焦虑，而没有降低认知参与度。根据Sutton等人的研究，替代互动的好处依赖于学习风格，那些通过替代互动积极思考的人通常在同步课堂中参与直接互动时感到不舒服[59]。尽管本研究没有可以比较的同步课堂条件，许多参与者提到SVD让他们在问题上积极思考，和使用EdPuzzle时一样，同时感到压力和焦虑都减少了。这种感受在P35的以下引用中得到了最好的体现：

“这确实鼓励我尝试思考和参与，而不是仅仅等待答案。这让我感觉几乎像我是学生，助教的提示鼓励我思考并扩展我的回答，因此变得更加难忘，因为我必须深入思考这些内容。感觉非常真实，就像助教或讲师在教育环境中所做的一样……我实际上感觉自己参与了对话，我认为这是一个很好的方式让学生参与进来，也对那些害羞或自信心不足的个人来说是一个巨大的优势——这压力更小，少了‘当场反应’的感觉，但同时也让你感到参与其中。”

最后，我们发现参与者在EdPuzzle中对问题的思考更加积极，但在SVD中对答案的思考更加积极。虽然两个系统都以问答格式呈现知识，但EdPuzzle迫使参与者回答问题，因此参与者对问题进行了深入思考，然后记住了答案，但没有在此基础上进一步积极思考（P37，P41，P47）。P43解释说，这是因为“在收到信息之前，我已有机会思考它所强化的观点。”另一方面，在SVD中，参与者并不被迫回答问题，因此许多人立即查阅答案。然而，由于答案是逐步构建成对话，他们“看到同学的发言，并反思自己的答案和信息”（P44）。P38表示，这可能是替代学习的另一个好处，“作为观察者让我有空间思考和分析他人的回应。”这一观察说明了教师的目的，无论是强调问题还是答案，都可能影响SVD在应用环境中的有效性。

# 6 讨论

# 6.1 结果解释及其意义

通过本研究，我们发现SVD对特定学习者群体的参与度产生积极影响，即那些重视人际互动、对虚拟角色感到舒适并喜欢主动控制学习节奏的学习者。量化结果显示，这些特征的学习者由于体验到的社会联系感和虚拟角色的相关性，与SVD体验到高度的情感参与。这些学习者在SVD中的行为参与也有所增加。这个增加是实证证据，表明替代互动理论可能适用于通过虚拟角色进行替代学习。它也支持了我们的假设，即行为参与是可以从直接互动者转移到替代互动者的一个好处，并且完善了我们在第3.1节提到的替代互动理论中的差距。虽然我们没有发现认知参与的显著差异，但我们看到证据表明SVD减少了表现焦虑，并可能在某些情境中导致更积极的思考。最后，我们看到更喜欢SVD的学习者可能会在此中获得更高的学习收益，作为第二级效果。

虽然我们在第5节中将影响EdPuzzle和SVD偏好的因素以二元变量的形式呈现，但这两个系统实际上是一个光谱的两个端点。对于每个因素，都可以调整SVD参数或添加新的属性以适应学习者的需求。

简洁性与人际互动：根据学习者的学习风格或情况，他们应该能够选择学习的展示方式。SVD不应作为一个单独的系统，而应作为呈现问答的替代方案，整合到已经具有更简单展示形式的系统中（例如，EdPuzzle）。此外，中间选择可以包括利用社会互动来增加教育参与度的现有作品，例如与虚拟助教的直接问答[42]，或仅用一个虚拟学生进行的替代问答[43]，等等。角色的虚拟性：许多参数可以影响一个人对虚拟性的感知以及虚拟性对学习者的影响。以往研究表明，互动特征，如目光[25]、语音模式[50]、手势和举止[35]，可以增强虚拟代理人的社会存在感和用户对代理人的信任。虽然SVD不是基于代理人的系统，但我们的数据显示，许多享受SVD的参与者表示他们“喜欢”某些角色。为了增加角色的相关性和可爱性，未来的工作可以考虑为角色添加互动属性。然而，我们也需要注意遵循DG1，避免让角色过于真实。我们的数据中另一个重要考虑是，某些学习者可能会将互动的情感或社交属性视为学习过程的障碍。一个明确的例子是课堂开始时的社交闲聊，这类反馈显示出两极化。我们建议，除了角色的身份和互动频率外，学习者应能够自定义每一个虚拟属性，而不是让其成为体验的固定组成部分。

•通过对话进行节奏控制：自我节奏学习已被证明能提高学生的参与感和元认知控制，尤其是在观看讲座视频等传统被动学习环境中[6, 45]。我们的研究结果与现有文献一致。具体来说，参与者表示自我节奏功能很好地满足了不同阅读速度学习者的需求。然而，我们也发现，过于细致的节奏控制可能会导致某些参与者的认知过载和 disengagement。在此基础上，我们建议改进交互方案，使得在每个节点，学习者可以选择是否打开一条新消息或自动播放线程，并且他们可以随时暂停自动播放，返回逐条点击。

# 6.2 替代对话与交互式虚拟代理

如第1节提到的，SVD专注于对话呈现，与根据学生行为和表现进行模拟对话的互动代理不同。因此，SVD的一些有效特征并不专属于教师编写的脚本，也可以应用于虚拟代理，以更具吸引力地“结构”和“呈现”教学互动。这些特征包括：

•替代学习：根据我们在认知参与方面的定性研究结果（第5.3节），替代学习的被动性质对教学思维过程（例如，思考构建答案的每一步，看到可能错误的方向）和学习态度（例如，回答错误是可以的）非常有用。可教代理也可以作为虚拟学生来向学习者展示这些行为。

教师的控制：除了信息隐私问题外，代理可能会做出有偏见的评论或以低效的方式传达内容，而教师已经掌握了内容并有经验如何有效地传递给学习者（例如，突出哪些错误，提问哪些引导性问题）。虚拟代理还可以利用教师的这些知识，更有效地传递课程。

•多人课堂环境：我们在5.1节中的研究结果表明，拥有多人有利于呈现多种视角并创造社会联系感。然而，设计多种角色是一个困难的社会设计问题，因为设计师必须关注由于多重身份、刻板印象和个性特征的互动而产生的群体动态，正如我们在3.3.2节中所介绍的。这些指导原则可以应用于虚拟代理的设计，以及社交机器人和多机器人系统的设计[57]。

# 6.3 编写有效对话

尽管SVD的设计对结果有贡献，正如我们在第5节中解释的，但不可否认的是剧本的质量也会影响结果。在本节中，我们将讨论剧本对参与者对SVD的感知的影响，以及撰写优质SVD剧本的挑战。需要注意的是，为了确保比较的公平性，两个条件下每段对话/问答的内容和时长是相同的。因此，我们将只讨论剧本的散文质量，而不是其内容。

我们发现脚本主要影响参与者对互动的自然感知，这是情感参与的一个因素。过于正式的语言（“学生提问的方式似乎比我在Zoom环境中实际观察到的要夸张得多，甚至在严格的大学课程中也是如此。” - P43）和虚假的友好（“试图让一些虚假变得更人性化[..]他们只会感到尴尬和笨拙。” - P21）会使虚拟角色对于一些学习者显得特别不自然。这些因素是影响聊天机器人或语音用户界面自然感的因素子集[37]，但需要额外考虑多个人物之间独特的社交动态。

我们花了很长时间修订每个视频的脚本，我们意识到对话脚本有很多维度。最重要的两个维度是每个对话气泡的简洁性和讨论的长度。

简洁性：我们希望每个聊天气泡都简洁明了，以便学习者能够轻松掌握知识，而不会因为填充词（例如“嗯，实际上。”或“我有点想...”）而分心。然而，我们也希望让其生动自然。找到合适的平衡是一个具有挑战性的问题，这取决于对话的内容和目的。

•讨论长度：我们希望通过多个聊天气泡进行反复讨论，以进行知识构建。但助教在揭示答案之前应该提出多少个引导性问题？虚拟学生应该自行弄清楚多少，虚拟助教又应该告诉他们多少？长时间的讨论会帮助学习者全面构建知识，但也会分散学习者对讲座的注意力。这种平衡也很难确定，与上述简洁性问题类似。

# 6.4 学生对SVD真实性的认知

除了DG1所规定的角色表现的真实性，另一个可能影响学生对SVD真实性感知的变量是对话内容的来源。也就是说，尽管学生们始终意识到对话是由讲师编写的，但他们仍可能怀疑讲师的剧本是基于他们之前授课班级的真实对话。这个微妙的变量在我们的设计中未得到解决，但它引发了伦理和实际的担忧。

伦理考虑在于感知的真实性是否会造成欺骗。在我们的参与式设计中，许多教师表示他们会使用SVD来呈现来自以往课堂上常见或有趣的问题。一方面，问题可能确实来自真实的学生，但另一方面，这些问题的集合并不能反映一个真实课堂，因为它们是由教师有意编制的。因此，我们不能直接应用现有框架[10, 65]来判断SVD是否具有欺骗性。我们希望通过通知学生脚本是由教师编写的（如第3.4节所述），学生至少能意识到教师对对话的潜在影响，而不相信这些对话是过去课堂的完整复制。

在实践方面，考虑这一变量如何影响学生的参与感和自我效能感至关重要。这两个价值观之间存在张力。一方面，学生对对话的真实性感知越高，他们的参与度（例如兴趣、注意力）就可能越高 [48]。另一方面，由于教师故意为剧本选择有趣或聪明的问题，学生对对话的真实性感知越高，他们的自我效能感就可能越低 [19]。尽管我们在设计SVD时尝试折中这两种对立的效果，但我们并没有测量个别学生对对话真实性的感知、他们的参与度和自我效能感之间的关系。我们建议将其作为未来研究的潜在领域。

# 6.5 开发创作工具的需求和可行性

在前一部分中提到的挑战下，我们看到编写脚本对于讲师来说可能是费力且耗时的。在我们的参与设计研究中，讲师对SVD的创作过程表示担忧。将现有的问答转化为自然流畅的对话需要时间，而他们不愿意抽出这样的时间。为了使SVD切实可行，它需要一个创作工具来缓解这些顾虑。

# 6.6 限制

本研究有三个主要局限性：视频、测验和脚本。视频全部集中在一个主题上，世界历史，这可能会影响结果的普遍性。具体来说，与不同学科有关的不同类型知识和知识活动可能会在学习者中引发不同类型的认知参与。例如，学习者在学习世界历史时主要进行记忆，而在学习数学时则更多进行应用。此外，视频长度也非常短，因此参与者可能没有机会忘记或与内容脱离。关于测验，每个测验中的题目数量较少，因此无法捕捉到学习收益中的重大差异，而多项选择题也可能引入测量学习收益的噪音。最后，如第6.3节所述，与SVD的参与部分依赖于脚本的质量，但我们无法量化这种影响，假设两个视频的脚本具有相同的质量和有效性。

# 7 结论

在本文中，我们提出了将奇异值分解（SVD）作为一种技术，旨在通过在教育视频周围编写虚拟助教与学生之间的模拟对话来增加孤立视频学习者的参与感。我们基于与教师的参与设计以及来自学习者的初步反馈来设计SVD。我们的评估研究表明，SVD的效果取决于个体对社交联系的需求、虚拟感的感知和学习策略。相比于EdPuzzle，偏好SVD的学习者在情感和行为参与度上显著更高。SVD还促进了对批判性话语的认知参与。我们将SVD与其他社交视频观看和替代学习体验进行了定位，并讨论了SVD的洞察如何被推广以改善其他增强视频学习的方法。未来的研究应探索SVD可能带来益处的其他情境，并为教师设计一个有效的SVD创作工具。
